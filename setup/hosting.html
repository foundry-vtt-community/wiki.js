Hosting

Methods of hosting and setting up networking so you can get started with Foundry VTT!

Please note that most methods here will not be officially supported by Foundry VTT, and you are unlikely to get any help if you ask on the official Discord server. Any guides on this wiki are followed at your own risk, and may be out of date.

Each list is in no particular order, and you should read the whole page before starting any guide.

Official Resources

Please see the official website's articles before returning here. The methods and options here will always be better supported and often easier to get up and running.

Officially supported hosting options: https://foundryvtt.com/article/hosting/

Installation guide: https://foundryvtt.com/article/installation/

Port Forwarding, for self hosting: https://foundryvtt.com/article/port-forwarding/

Self-Hosting

These guides are for installing and setting up Foundry on your own hardware. There is some crossover with cloud hosting options, especially when tools like Docker are involved.

  • Ubuntu VM - full guide including setting up a custom domain name using nginx and SSL
  • Arch Linux
  • Docker - multiple different dockerfiles with varying levels of complexity and features
  • FreeBSD
  • Android Phone
  • Kubernetes
  • Local Play - options for local in-person playing with minimal setup
  • Raspberry Pi - outdated, use the official linux setup instructions instead
  • Synology NAS
  • Synology with Docker
  • TrueNAS SCALE

Escaping Restrictive Networks

If port forwarding won't work for you, some of these options might. They are in no particular order, and are used at your own risk. Some may expose your computer to the internet in ways that bypass firewalls that keep you safe.

All of these options will require having already installed and set up Foundry VTT.

  • Hamachi - software that creates a virtual local network between up to 5 people with Hamachi installed. Page WIP, see section in Windows Setup for now.
  • ZeroTier - similar to Hamachi without a connection number limit. Page WIP, see section in Windows Setup for now.
  • CloudFlare Proxying and Tunnelling
  • Tailscale Funnel - free WireGuard based forwarding, HTTPS without extra setup, no installation needed for players
  • LocalToNet
  • ngrok
  • Apache Reverse Proxy
  • DuckDNS
